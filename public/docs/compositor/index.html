The Compositor

The Greenfield compositor is what you see in the browser when you start the Greenfield cloud desktop. It's main responsibility is drawing client applications, along with providing the means to manage them.

Get the sources:

git clone https://github.com/udevbe/greenfield.git

Inside the compositor directory run

  npm install

To start the compositor in development mode, run

  npm run start

And wait for the a browser to automatically open a tab to localhost:8080.
